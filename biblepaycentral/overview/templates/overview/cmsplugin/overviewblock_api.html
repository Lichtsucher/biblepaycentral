APIs

Every modern Blockchain has a lot of 3rd-Party APIs, and Biblepay is no different. Dig deep into these and access the full power of biblepay.

  • Biblepay Client
    Use the official Biblepay commandline client (biblepay-cli) to access all the details of the Biblepay Blockchain or use the RPC Interface of the daemon for your own services.

  • Explorer API
    The Biblepay-Central Explorer API offers some important webbased APIs, to understand and analyse the Biblepay Blockchain easily as REST/JSON Webservice.

    https://explorer.biblepay-central.org/info
